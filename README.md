# COVID-19 Data Analysis using Python

## Project Overview
This project performs exploratory data analysis on COVID-19 dataset using Python. The goal is to analyze global COVID trends, affected countries, and relationships between confirmed, death, and recovery cases.

## Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
The dataset is in CSV format and contains information about:

- Date
- Country/Region
- Province/State
- Confirmed Cases
- Deaths
- Recovered Cases

## Project Steps

1. Import required libraries
2. Upload dataset in Google Colab
3. Load dataset using Pandas
4. Explore dataset structure
5. Clean missing values
6. Convert date column
7. Group data by country
8. Identify top affected countries
9. Visualize results using graphs
10. Analyze COVID trends over time
11. Generate correlation heatmap

## Visualizations Included

- Bar chart of top 10 affected countries
- Line chart showing COVID trend over time
- Scatter plot of deaths vs recoveries
- Correlation heatmap

## Conclusion

This project helps understand COVID-19 spread patterns and relationships between confirmed, death, and recovered cases using data visualization and analysis techniques.
